# List of Coding Challenge Sites

### Site Counter: 80+

This is a **collection** of coding challenge sites, CTF sites, and interviewing resources that are useful for **learning and practicing coding**, or preparing for coding interviews.

This list is long, so it's divided into **Groups A to E**, each having a **maximum of 15 sites**. The sites in each group are **sorted by popularity**, with the most popular ones at the top for easier search.

These sites are trusted to ensure **no malicious link is included** (in case of Privacy Errors). New Sites are updated in a **group that is not full**, or be kept at **different categories**.



# Site Categories

## Coding Challenge Sites

These sites are used for **practicing problems** that can help you master the fundamentals from the problemsets. You can use them for grinding (for fun), learning how to code, or just preparing for coding interviews.

### Group A

1. **[Leetcode](https://leetcode.com)** - The most popular platform for coding interview preparation. It has thousands of problems ranging from easy to hard, with a strong focus on data structures and algorithms. Companies often use LeetCode problems in their technical interviews, making it essential for job seekers.

2. **[HackerRank](https://www.hackerrank.com)** - A widely-used platform that offers coding challenges, competitions, and company-specific hiring contests. It covers multiple domains including algorithms, databases, and artificial intelligence. Many companies use HackerRank to screen candidates.

3. **[GeeksforGeeks](https://www.geeksforgeeks.org)** - A comprehensive computer science learning platform that combines tutorials with practice problems. It has detailed explanations, articles, and interview experiences from top tech companies. Great for learning concepts and seeing problem solutions.

4. **[CodeForces](https://codeforces.com)** - One of the most popular competitive programming platforms globally. It hosts regular contests featuring tough algorithmic problems. Known for its fast-paced contests and strong community of competitive programmers. Ideal for improving problem-solving speed.

5. **[CodeChef](https://www.codechef.com)** - A competitive programming platform that hosts monthly contests and provides a large problem library. It has a rating system and supports multiple programming languages. Good for those looking to compete in programming contests regularly.

6. **[CodeWars](https://www.codewars.com)** - A gamified platform where you solve coding challenges called "katas" in various languages. It focuses on improving coding skills through repetition and mastery. Great for learning new languages and improving code efficiency.

7. **[HackerEarth](https://www.hackerearth.com)** - A platform that combines coding practice with hiring challenges. It offers problems across different difficulty levels and hosts hackathons. Many companies use it for recruitment and talent assessment.

8. **[InterviewBit](https://www.interviewbit.com)** - An interview preparation platform with curated problems similar to those asked in top tech companies. It follows a structured learning path and tracks your progress. Great for systematic interview preparation.

9. **[NeetCode](https://neetcode.io)** - A LeetCode-style platform specifically designed for interview preparation. It offers curated problem sets organized by company and difficulty. Popular among developers preparing for FAANG interviews.

10. **[CodinGame](https://www.codingame.com)** - A unique platform that teaches programming through games and puzzles. You write code to control game characters and solve challenges. Fun way to practice coding in over 25 programming languages.

11. **[SPOJ](https://www.spoj.com)** - Sphere Online Judge is an online judge with a massive archive of problems. It has thousands of problems from various categories and supports many languages. Great for intensive practice with a large problem bank.

12. **[Edabit](https://edabit.com)** - An interactive platform offering bite-sized coding challenges for beginners. Problems are designed to teach programming concepts step by step. Perfect for those just starting their coding journey.

13. **[CodingBat](https://codingbat.com)** - Simple practice problems in Java and Python, focused on basic logic and recursion. Created by Nick Partridge at UC Berkeley. Good for warming up with simple coding exercises.

14. **[GreatFrontend](https://www.greatfrontend.com)** - A platform focused on frontend development practice and interview preparation. Covers JavaScript, React, CSS, and system design for frontend roles. Great for frontend developer interviews.

15. **[AlgoMonster](https://algo.monster)** - An algorithms and data structures focused platform with a structured curriculum. It emphasizes pattern recognition and efficient problem-solving techniques. Popular for systematic algorithm practice.

## CTF & Security Platforms

CTF (Capture The Flag) sites are used for **practicing cybersecurity skills** and **solving challenges**. They are useful for learning about cybersecurity, penetration testing, and ethical hacking, while also providing a platform for competitive programming and problem-solving. There are also **Cryptography sites** for learning and practicing cryptography concepts and techniques.

1. **[TryHackMe](https://tryhackme.com)** - A beginner-friendly platform for learning cybersecurity through guided hacking labs. It offers step-by-step rooms and challenges for various skill levels. Perfect for those starting their cybersecurity journey.

2. **[HackTheBox](https://www.hackthebox.eu)** - An advanced penetration testing practice platform. It offers challenging machines and CTF-style challenges. Popular among security professionals and enthusiasts.

3. **[PicoCTF](https://picoctf.org)** - A beginner-friendly CTF competition primarily for high school and college students. Problems range from easy to moderate difficulty. Great introduction to CTF competitions.

4. **[OverTheWire](https://overthewire.org)** - A series of wargames for learning and practicing security concepts. Starts from very basics and progresses to advanced topics. Excellent for self-learning security.

5. **[CTFLearn](https://ctflearn.com)** - A community-driven platform where users can practice CTF challenges. Offers problems in various categories. Good for beginners to advanced practitioners.

6. **[CryptoHack](https://cryptohack.org)** - A platform for learning cryptography through challenges. Covers modern cryptography concepts and techniques. Great for learning and practicing crypto skills.

7. **[W3Challs](https://w3challs.com)** - A security challenges platform covering web, system, and network security. Offers challenges in multiple categories. Good for comprehensive security practice.
